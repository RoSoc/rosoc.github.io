Buddhabrot Fractal

Buddhabrot Fractal rendering program built in haskell for a University contest.

This represents a program rendering a Buddhabrot fractal using Haskell language and OpenGL libraries. This Project was developed for a Fractal development competition held at University level in First Academic Year's Functional Programming course. Flappy Bird for Desktop

Flappy Bird clone for Desktop built in Java for a first-year OOP project.

This represents a program developed for an optional project in my first academic year with the purpose of practising object oriented programming skills in Java. It actually represents a clone of the well known Flappy Bird game, built using LWJGL graphics libraries (used in Minecraft as well). This project also contains an Android-compatible version and the entire program was developed in eclipse using the additional Android SDK. For the desktop version to work Android SDK is not necessary.

  • Category: Java / Object Oriented Programming

Cloth Engine

Cloth engine built in Javascript using WebGL.

This project represents a cloth engine built using Javascript and WebGL, mostly the requestAnimationFrame() function. The cloth is drawn using a HTML5 canvas. The project on GitHub also contains a gh-pages branch, so a demo of the program is available here.
